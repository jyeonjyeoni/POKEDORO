# POKEDORO Web v1.2.4

집중한 시간만큼 탐험하고 포켓몬 친구를 만나는 뽀모도로·타이머 웹앱입니다. 기존 Windows 데스크톱판의 게임 데이터 구조와 주요 기능을 브라우저/PWA 환경에 맞게 옮겼습니다.

## 주요 기능

- 항상 보이는 컴팩트한 메인 위젯과 탭형 기능 화면
- 사용자 지정 뽀모도로, 카운트업 타이머, 자동 집중·휴식 전환
- 백그라운드 탭/절전 복귀 시 실제 경과 시간 정산 및 완료 알림
- 집중 30분마다 탐험권 1장, 최대 3장 보유
- 카테고리, 예상 시간, 수정, 다중 완료·삭제, 드래그 우선순위를 지원하는 할 일
- 장소 시드 기반 포켓몬 종·성격 재현과 별도 이로치 판정
- 장소 시드와 독립된 55%·35%·10% 폼 판정 및 1,403개 모습 수집 도감
- 반응 애니메이션과 성격 힌트가 있는 친해지기 미니게임
- 친구 검색·도감번호순 정렬, 최대 5마리 방 배치, 친밀도, 놓아주기 확인, 친밀도 100 진화
- 진화 차단용 변함없는돌과 탐험권 30장으로 해금하는 자동 쓰다듬기 기계
- PokeAPI 전국도감 1,025종, 개체별 폼 스프라이트와 한국어·영어·일본어 이름
- 도트/HOME/공식 이미지, 5가지 컬러 테마, 정적 모드, 볼륨·음소거
- 사용자 배경 이미지, 메인 화면 PNG 클립보드 복사
- IndexedDB 자동 저장, 최근 14일 일일 백업, JSON 내보내기·가져오기
- 설치 가능한 PWA와 오프라인 앱 셸·데이터 캐시

## 실행

```bash
npm install
npm run dev
```

프로덕션 검증:

```bash
npm test
npm run build
```

빌드 결과는 `release-dist/`에 생성됩니다.

## 백그라운드 동작

브라우저가 백그라운드 탭의 JavaScript 실행을 늦추더라도 저장된 기준 시각으로 집중·휴식 구간을 다시 계산합니다. 알림 권한을 허용하면 완료 알림을 표시하며, 앱 탭 또는 설치형 PWA가 살아 있는 동안 종료음을 재생합니다. 브라우저나 운영체제가 앱 프로세스를 완전히 종료한 상태에서는 웹 보안 정책상 즉시 소리 재생을 보장할 수 없지만, 재실행 시 경과 시간과 보상은 복원됩니다.

## 데이터와 출처

사용자 데이터는 서버로 전송하지 않고 해당 브라우저의 IndexedDB에 저장합니다. 포켓몬 정보와 이미지·울음소리는 [PokeAPI](https://pokeapi.co/) 및 PokeAPI 공개 자산을 사용합니다.

POKEDORO는 비공식 팬 프로젝트입니다. Pokémon 및 관련 명칭·이미지의 권리는 각 권리자에게 있습니다.
